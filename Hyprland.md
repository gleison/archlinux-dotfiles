# Hyprland

Install minimal packages

```bash
sudo pacman -S hyprland hyprlock hypridle waybar kitty wofi dolphin dolphin-plugins ark
```

description:
**hyprland** the wayland compositor
**hyprlock** lock screen for hyprland
**hypridle** idle for hyprland
**waybar** status bar
**kitty** terminal emulator default for hyprland
**wofi** laucher default for hyprland
**dolphin** file explorer default for hyprland
**dolphin-plugins** plugins for dolphin
**ark** compressed file utility

## First configuration


Edit de hyprland.conf file and cofigure:

- Keyboard
- Monitor Resolution
- Remove auto-generated warning
